Jogo do Galo - Daniel Silva

Descrição:

O projeto consiste num jogo do galo baseado na web em que os jogadores podem arrastar e largar uma cruz ou um círculo num tabuleiro de jogo 3x3. O jogo inclui um painel de avaliação para registar as pontuações de ambos os jogadores (Jogador X e Jogador O). O vencedor é determinado com base nas regras tradicionais do jogo do galo e é apresentada uma mensagem quando um jogador ganha.

Instalação
Copie o link para aceder ao repositório no GitHub e descarregue os ficheiros (index.html e style.css)

Como jogar:

Abra o ficheiro HTML (index.html) no browser;
Arraste e largue os símbolos da cruz ou do círculo no tabuleiro de jogo 3x3.
Após alguém ganhar o painel de avaliação no topo da página será atualizado com base no vencedor.

Características:

Funcionalidade de arrastar e largar para colocar símbolos no tabuleiro de jogo.
Painel de avaliação para registar as pontuações do jogador X e do jogador O.
Deteção de vitórias com base nas regras tradicionais do jogo do galo.
